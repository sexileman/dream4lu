## 介绍
1. 使用 [VuePress v2.0](https://github.com/vuepress/vuepress-next) 搭建的个人博客

## 快速上手

```bash
# clone the project
git clone https://github.com/sexileman/dream4lu.git

# enter the project directory
cd dream4lu

# install dependency
npm install # or yarn install

# develop
npm run dev # or yarn dev
```
